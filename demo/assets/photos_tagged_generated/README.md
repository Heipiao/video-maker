# Tag-First Generated Photo Set

This folder contains a generated demo set for `nostalgia_love_story_40`.

The images were created from the tag-first plan in:

`demo/data/image_generation_plan.demo.json`

Each source generation produced a three-panel wedding triptych. The panels were cropped into `photo_001.jpg` through `photo_012.jpg`.

Use this set for local template testing only. Production jobs should use user-uploaded images or a separately licensed seed dataset.
